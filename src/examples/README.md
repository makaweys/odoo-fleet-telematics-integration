# API Examples and Test Requests

This folder contains example requests for testing the Vehicle Telematics Tracker API.

## Files:

1. **vehicle-examples.md** - Vehicle management endpoints
2. **location-examples.md** - Location/Traccar endpoints  
3. **odoo-examples.md** - Odoo integration endpoints
4. **geofence-examples.md** - Geofence management
5. **poi-examples.md** - POI (Point of Interest) management
6. **zone-examples.md** - Zone management
7. **react-frontend.js** - React frontend API calls
8. **curl-requests.sh** - All curl commands in one file

## Quick Start:

```bash
# Run all curl examples
./curl-requests.sh

# Or run specific sections
bash curl-requests.sh --vehicles
bash curl-requests.sh --location
